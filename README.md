# 2XBET
Batak, Blackjack, Rulet ve slot oyunlarından oluşan 2XBET c# ile yazılmış bir form uygulaması oyunudur. Oyun Algoritmaları tamamen bana aittir. Yaptığım ilk büyük proje olduğu için Kod standartlarına pek uygun değil yani clean code değil.
